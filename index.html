⚙️ Configuração

🔑 Chave API Groq (IA Gratuita)
Obtenha sua chave gratuita em console.groq.com → API Keys → Create API Key. Totalmente gratuito.
☁️ Google Drive
Gratuito, sem cobrança e sem OAuth. Um script salva os arquivos direto no seu Drive, organizados por turma e bimestre.
Passo 1 — Criar o script
  1. Acesse script.google.com e clique em Novo projeto
  2. Apague todo o código e cole o código abaixo:
  1. Clique em Implantar → Nova implantação
  2. Tipo: App da Web  |  Executar como: Eu  |  Acesso: Qualquer pessoa
  3. Clique em Implantar, autorize e copie a URL do app da Web
⚠️ Usuários veem "Você precisa ter acesso" ao abrir documentos?
O script precisa ser atualizado para compartilhar os docs automaticamente. Siga os passos:
  1. Acesse script.google.com → abra o projeto
  2. Apague o código atual e cole o código novo acima (já corrigido) — clique em 📋 Copiar
  3. Clique em Implantar → Gerenciar implantações → lápis ✏️
  4. Escolha Versão: Nova versão → Implantar
  5. Se pedir autorização, aprove tudo
Após isso, novos documentos serão criados já com permissão de acesso para todos.

Documentos antigos (já criados sem permissão): abra cada um no Drive → clique em Compartilhar → Qualquer pessoa com o link.
Passo 2 — Conectar o app
📁 Estrutura no Drive: Escola Ponte do Saber / {Turma} / {Bimestre} / Pareceres|Planejamentos
